# Aplicação NodeJS
Simples aplicação feita em NodeJS para realizar requisições em uma aplicação feita em PHP.

 ## Pré-requisitos
 - NodeJS 8.10+

## Instruções

 - Execute o comando `cp .env.example .env`
 - Altere as informações do arquivo `.env`
 - Rodar `npm install`
